### Словник атрибутів об'єктів

| Об’єкт                 | Атрибут           | Короткий опис                                           | Тип        | Обмеження             |
|------------------------|-----------------|--------------------------------------------------------|-----------|----------------------|
| User             | full_name             | Ім'я та прізвище користувача                                       | текст     | не більше 50 символів         |
|   User           | user_role             | Посада співробітника               | текст     | не більше 30 символів         |
| Workplace            | description             | Опис робочого місця (комп’ютер, освітлення, ергономіка) | текст     | не більше 200 символів         |
| WorkplaceAssessment    | assessment_date             | Дата проведення оцінки                                 | дата      | формат дд.мм.рррр         |
|  WorkplaceAssessment   | assessment_result        | Результат оцінки                                       | текст     | не більше 300 символів         |
| LightingLevel      | light_level          | Рівень освітленості на робочому місці                  | число  | не більше 100   |
| SafetyRisk            | risk_type       | Тип потенційного ризику             | текст     | за замовчуванням "Немає ризиків", максимум 50 символів       |
|   SafetyRisk          | risk_details       | Опис деталей потенційного ризику            | текст     | не більше 250 символів       |
| Recommendation            | recommendation_text           | Поради щодо покращення умов праці                     | текст     | не більше 250 символів         |
| VideoConference        | topic             | Тема відео-конференції                                      | текст     | не більше 100 символів         |
|    VideoConference     | start_time       | Час початку конференції                                | timestamp  | формат дд.мм.рррр, ГГ:ХХ         |
| Participant                 | full_name    | Ім'я учасника відео-конференції                        | текст     | не більше 50 символів         |
