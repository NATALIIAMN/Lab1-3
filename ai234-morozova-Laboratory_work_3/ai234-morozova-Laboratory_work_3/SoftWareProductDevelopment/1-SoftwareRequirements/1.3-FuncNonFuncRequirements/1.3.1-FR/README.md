### Опис функціональних вимог

| FR      | Опис функціональної вимоги                             |
| ------- | ------------------------------------------------------ |
| **FR2** | Оцінити робоче місце                                   |
| FR2.1   | Користувач запускає процес оцінки робочого місця       |
| FR2.2   | ПП аналізує рівень освітленості робочого простору      |
| FR2.3   | ПП визначає потенційні ризики для безпеки користувача  |
| FR2.4   | ПП формує рекомендації для покращення умов праці       |
| FR2.5   | Користувач переглядає результати оцінки робочого місця |
| FR2.6   | Забезпечується безпека та комфорт під час роботи       |
| **FR3** | Провести відео-конференцію з колегами                  |
| FR3.1   | Користувач створює відео-конференцію                   |
| FR3.2   | Користувач додає учасників конференції                 |
| FR3.3   | ПП забезпечує підключення учасників у реальному часі   |
| FR3.4   | Користувач проводить обговорення робочих питань        |
| FR3.5   | Користувач завершує конференцію                        |
| FR3.6   | Досягається мета – ефективна комунікація з колегами    |
