### Опис вимог до INPUT-потоків інтерфейсу користувача

| FR    | NFR    | Засіб INPUT-потоку           | Особливості використання                           |
|-------|--------|-----------------------------|---------------------------------------------------|
| FR2.1 | NFR2.1 | Кнопка миші / сенсорний екран | Користувач натискає кнопку “Почати оцінку”       |
| FR2.2 | NFR2.2 | Клавіатура / миша            | Проводиться оцінка безпеки праці за параметрами: рівень освітленості, відстань до монітора, рівень шуму, температура приміщення          |
| FR2.3 | NFR2.3 | Сенсорний екран / миша        | Користувачу виводиться на екран список з потенційних ризиків для його безпеки                    |
| FR2.4 | NFR2.4 | Миша / клавіатура            | Користувач переглядає сформовані рекомендації для усунення ризиків                |
| FR2.5 | NFR2.5 | Кнопка миші / клавіатура            | Користувач завершує процес оцінки                 |
| FR3.1 | NFR3.1 | Кнопка миші / сенсорний екран | Введення нікнейму користувача, назви та теми конференції                             |
| FR3.2 | NFR3.2 | Клавіатура / миша            | Вибір учасників конференції через чекбокси, натискання кнопки "Підтвердити вибір"                               |
| FR3.3 | NFR3.3 | Система підключення          | Автоматичне встановлення зв'язку з вибраними учасниками                  |
| FR3.4 | NFR3.4 | Мікрофон / клавіатура / камера | Обмін повідомленнями і голосове спілкування          |
| FR3.5 | NFR3.5 | Кнопка миші / сенсорний екран | Натискання кнопки "Завершити конференції"                             |
