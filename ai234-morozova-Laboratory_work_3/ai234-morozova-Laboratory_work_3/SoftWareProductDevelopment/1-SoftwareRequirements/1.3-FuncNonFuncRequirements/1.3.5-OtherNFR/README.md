### Опис основних нефункціональних вимог, не пов'язаних з інтерфейсом користувача

| FR    | NFR      | Тип NFR                   | Опис NFR                                                                 |
|-------|----------|---------------------------|-------------------------------------------------------------------------|
| FR2.1 | NFR2.1   | Time Performance          | Процес запуску оцінки робочого місця має виконуватись не більше ніж за 5 секунд після натискання кнопки "Почати оцінку"                 |
| FR2.2 | NFR2.2   | Hardware Interface        | Програма аналізує рівень освітленості, шуму робочої зони за допомогою сенсорів         |
| FR2.3 | NFR2.3   | Hardware Interface        | Аналізуються параметри для визначення усіх ризиків для безпеки праці         |
| FR2.4 | NFR2.4   | Software Interface        | Система на пристроях Windows та Andoid отримує дані від сенсорів освітленості через REST-запит у форматі JSON та забезпечує безпечний доступ користувачів через HTTPS і токени                       |
| FR2.5 | NFR2.5   | Human Factors Requirements| Результати оцінки відображаються у зручному адаптивному інтерфейсі                   |
| FR2.6 | NFR2.6   | Design Constraints        | Забезпечується відповідність безпековим стандартам робочого місця                            |
| FR3.1 | NFR3.1   | Time Performance          | Створення нової відео-конференції виконується за ≤ 3 секунди після підтвердження користувачем                                          |
| FR3.2 | NFR3.2   | Software Interface        | Система на пристроях Windows та Android дозволяє проводити відео-конференції в реальному часі з автентифікацією користувачів через токени та захищене з'єднання HTTPS                                   |
| FR3.3 | NFR3.3   | Human Factors Requirements| Інтерфейс відео-конференції є інтуїтивним і зрозумілим, підтримує вибір учасників, теми та швидкий старт зустрічі.          |
| FR3.5 | NFR3.4   | Design Constraints        | Завершення конференції має відбуватись без втрати даних та збереженням журналу сесії.                        |
