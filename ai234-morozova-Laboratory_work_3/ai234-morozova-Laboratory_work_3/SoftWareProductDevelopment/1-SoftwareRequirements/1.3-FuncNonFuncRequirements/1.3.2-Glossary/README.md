### Глосарій термінів

| Термін                     | Визначення |
|-----------------------------|-------------|
| **ПП (програмний продукт)** | Програмне забезпечення, що реалізує функціональні вимоги системи. |
| **Користувач**              | Особа, яка взаємодіє з програмним продуктом для виконання певних дій або отримання результатів. |
| **Робоче місце**            | Простір, у якому користувач виконує свої професійні обов’язки; може включати комп’ютер, монітор, освітлення тощо. |
| **Оцінка робочого місця**   | Процес аналізу умов праці (освітленість, ергономіка, безпека тощо) для визначення рівня комфорту та безпеки. |
| **Освітленість**            | Рівень світла на робочому місці, який впливає на зір і продуктивність користувача. |
| **Безпека користувача**     | Сукупність заходів, спрямованих на зменшення ризиків травмування або перевтоми під час роботи. |
| **Рекомендації**            | Автоматично сформовані поради від ПП щодо покращення умов праці користувача. |
| **Відео-конференція**       | Онлайн-зустріч користувача з колегами через відеозв’язок у реальному часі. |
| **Учасники конференції**    | Користувачі, запрошені до відео-конференції для обговорення робочих питань. |
| **Реальний час**            | Режим роботи ПП, у якому передача звуку, відео або даних відбувається без помітної затримки. |
| **Комунікація**             | Обмін інформацією між користувачами під час роботи або відео-конференції. |
