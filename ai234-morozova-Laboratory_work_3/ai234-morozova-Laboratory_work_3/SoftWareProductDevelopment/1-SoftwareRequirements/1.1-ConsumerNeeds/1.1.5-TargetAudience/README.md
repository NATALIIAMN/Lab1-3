### Цільова аудиторія споживачів програмного продукту

Програмний продукт розрахований на такі групи споживачів:

- **Національність:** не має значення; 
- **Стать:** не має значення;  
- **Вік:** повнолітні;
- **Рівень освіти:** вища;
- **Рівень доходу:** середній, вище середнього;
- **Зайнятість:** працює.
